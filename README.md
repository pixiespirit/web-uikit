# Web UIKit

The core UIKit for Pixie web applications.

## Development

Install package at root, /site and /core then

```
cd core
yarn run watch
```

```
cd site
yarn run dev
```

## @pixiepkg/core

Pixie web core UI components

### Install

```
yarn add @pixiepkg/core
```

## @pixiepkg/icon

Pixie icon sets for UI Kit

### Install

```
yarn add @pixiepkg/icon
```
